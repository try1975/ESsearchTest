	А. Формат csv - кавычки, разделитель полей, дробные цифры, отсутствие строки заголовка
	Б. Обязательные поля - порядок, описание
	В. Дополнительные поля - общие, частные. Модификаторы (латинские наименования, список модификаторов)
	Г. Если список модификаторов надо дополнить - порядок действий
	
	А. Для загрузки в elasticsearch через logstash csv файл должен соответствовать следующим требованиям:

	1. иметь расширение .csv
	2. иметь кодировку WIN1251 (CP1251)
	3. не должно быть строки заголовка с наименованием полей
	4. разделитель полей - точка с запятой(;). Ограничитель полей - двойная кавычка (") - внутри поля данных не должно быть двойных кавычек. 
	5. Разделитель строк - перевод строки (последовательность ASCII #13#10 - CRLF)
	6. Разделитель дробной и целой части чисел - запятая(,)
	
	Б. В файле должны быть обязательные поля в строгом порядке (на примере http://www.apteka-ifk.ru/art/32/)
	
	1. Url по которому располагаются данные "http://www.apteka-ifk.ru/art/32/" При переходе по этому url мы должны увидеть информацию из строки
	2. Наименование ТРУ "Л-Тироксин 50, тбл 50мкг №50"
	3. Цена "92,10"
	4. Наименование источника (как правило человеко-читаемое и краткое) "Аптека ИФК"
	5. Время сбора данных в формате UNIX-time - "1488379008"
	6. Валюта - "RUB" (ISO 4217) или "EUR" или "USD"
	Пример минимального набора обязательных полей в строке:
	
	"http://www.apteka-ifk.ru/art/32/";"Л-Тироксин  50, тбл 50мкг №50";"92,10";"Аптека ИФК";"1488379008";"RUB"
	
	В. В файле, кроме обязательных полей, могут присутствовать дополнительные. К общим дополнительным полям относятся:
	
	1. Телефоны поставщика с модификатором phones: , например "phones:84951112223 84992223337"
	2. Наименование производителя с модификатором manufacturer: , например "manufacturer:Berlin-Chemie/Menarini Group, Германия"
	2. Описание ТРУ, содержащее специфические именно для этого ТРУ сведения с модификатором description: , например "description:ЛЕВОТИРОКСИН НАТРИЯ (LEVOTHYROXINE SODIUM)"
	
	К частным дополнительным полям относятся:
	
	1. Сведения об упаковке с модификатором packaging: Например "packaging:50кг"
	
	Г. Если список модификаторов надо дополнить - пишите мне письмо на kf1975@mail.ru
